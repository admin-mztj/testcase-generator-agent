# 测试用例生成智能 Agent

一个基于多Agent协作的自动化测试用例生成工具，可自动解析需求文档，生成标准化测试用例并导出Excel。

## 功能介绍
- 解析TXT/MD格式需求文档
- 自动生成：功能测试用例、边界测试用例、异常测试用例
- 支持一键导出Excel
- 可视化GUI界面，零代码使用

## 核心Agent
1. 需求解析Agent
2. 用例生成Agent
3. 文档导出Agent
